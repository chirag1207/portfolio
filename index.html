StudentCheckin App

Abstract

The StudentCheckin App is an Android-based application designed to streamline event management for students. Users can sign up and log in to manage their personal profiles, view upcoming events, register for them, and track their registered events. The system ensures a seamless user experience by integrating a mobile frontend with a robust backend API.

Tech Stack Used

  • Frontend: Android (Java & XML), Volley (for HTTP requests)
  • Backend: Spring Boot (RESTful APIs)
  • Testing: Postman (API endpoint testing)
  • Database: MySQL or H2 (as per backend config)

Problems Faced

  • Handling asynchronous API calls with proper error handling using Volley
  • Ensuring secure user authentication and session handling
  • Integrating and syncing event registration state between frontend and backend
  • Debugging CORS and API access issues during integration testing
